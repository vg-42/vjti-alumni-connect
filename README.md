# VJTI Alumni Connect
A mobile application for alumni networking, built with React Native and Supabase.

---

# Features

- Authentication — Secure email and password sign up and login with persistent sessions
- Alumni Feed — Post updates, photos, and announcements visible to the entire network
- Comments — Comment on posts and engage with other members
- Likes — Like posts from other alumni
- Alumni Directory — Browse and search all members by name, batch year, or specialization
- Job Board — Post and browse job opportunities and referrals within the network
- Profile — Customizable profile with name, bio, graduation year, specialization, and profile photo
- Image Uploads — Attach images to posts and set a profile picture

---

# Tech Stack

- React Native (Expo)
- Supabase (Auth, Database, Storage)
- React Navigation

---

# Platform

Android

## 📝 Notes

- This is a college MVP project — not for production use
- No custom backend, no Express, no JWT handling — all handled by Supabase
- Keep logic in components or simple files for readability
